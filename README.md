# Calculadora de IMC 🏋️

Meu primeiro projeto em Python — calcula o Índice de Massa Corporal (IMC) e retorna a classificação de saúde.

## Como usar

1. Clone o repositório
2. Execute no terminal: python imc.py
3. Informe seu nome, peso e altura
4. Veja seu IMC e classificação

## Tecnologias

- Python 3
